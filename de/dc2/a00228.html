Make sure all the required data are presented.

  1. Check and get the main model database
  2. Check the existence of FILE_IN, FILE_OUT, PARAMETERS, REACHES, SITELIST, SPATIAL, etc

  4. Read Reaches data
  5. Read Mask raster data
