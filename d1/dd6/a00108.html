)

Calculates flow rate or flow velocity using Manning's equation. If x1 is set to 1, the velocity is calculated. If x1 is set to cross-sectional area of flow, the flow rate is calculated.

Parameters
[in]x1cross-sectional flow area or 1, m^2 or none.
[in]x2hydraulic radius, m.
[in]x3Manning's "n" value for channel.
[in]x4average slope of channel, m/m.
Returns
flow rate or flow velocity, m^3/s or m/s.
