42 ## i. pip安装
43 
44 + 离线安装pip
45 
46  离线安装pip需要下载setuptools和pip并解压
47 
48  随后，管理员方式运行cmd，输入以下命令：
49 
50  ```shell
51  cd <setuptools path>
52  python setup.py install
53  cd <pip path>
54  python setup.py install
55  ```
56 
57 + 在线安装pip，从[这里](https://bootstrap.pypa.io/get-pip.py)下载`get-pip.py`
58 
59  ```python
60  python get-pip.py
61  ```
62 
63 安装好pip，就可以离线安装whl格式的python第三方库了，whl库的下载地址为：[http://www.lfd.uci.edu/~gohlke/pythonlibs/](http://www.lfd.uci.edu/~gohlke/pythonlibs/)。
64 
65 ## ii. 使用pip安装whl依赖库
66 
67 + GDAL（依赖于Python）安装
68  通过上面下载whl库的链接，可以下载完整的GDAL打包文件 `GDAL-1.11.3-cp27-none-win_amd64.whl`(64位，注意32位和64位的区分)，管理员方式运行cmd，输入安装命令：
69 
70  ```shell
71  cd <GDAL path>
72  pip install GDAL-1.11.3-cp27-none-win_amd64.whl
73  ```
74 
75  提示安装成功即可，输入以下命令验证是否安装成功：
76 
77  ```python
78  python
79  from osgeo import gdal
80  from osgeo import ogr
81  from osgeo import osr
82  from osgeo import gdal_array
83  from osgeo import gdalconst
84  ```
85 + whl依赖库的批量安装
86  将下载的whl库文件(比如scipy，numexpr，PyQt4，wxPython（需要wxPython_common），PyGTK，python_dateutil，pytz，matplotlib，pywin32…)放到某一英文目录下，然后新建`install.bat`文件，输入如下内容：
87 
88 > 注意修改python路径：`PYPATH`
89 > install.bat文件应与whl库文件放在同一文件夹。
90 
91 ```shell
92 @echo off
93 @echo off
94 echo "Liangjun Zhu, zlj@lreis.ac.cn"
95 echo "-- Install Python Packages Using Pip..."
96 echo "-- WHL files downloaded from http://www.lfd.uci.edu/~gohlke/pythonlibs/"
97 pushd %~dp0
98 cd %~dp0
99 :: python path, x86 or x64 versioned python
100 set PYPATH=C:/python27x86/
101 set PIPPYTH=%PYPATH%Scripts/
102 :: COMMENT-1: Install easy_install and pip without network
103 ::%PYPATH%python %~dp0\setuptools-18.2\setup.py install
104 ::%PYPATH%python %~dp0\pip-7.1.2\setup.py install
105 :: COMMENT-2: Install easy_install and pip without network
106 ::%PYPATH%python get-pip.py
107 for /f "delims=" %%i in ('dir /s/b "*.whl"') do (
108 echo installing %%~ni ...
109 %PIPPYTH%pip install %%i
110 )
111 echo "-- All packages installed Succeed!"
112 pause
113 ```
114 
115 这样便会自动安装该文件夹所有的packages...

120 # 3. 最好用的Python编辑器之一--PyCharm介绍
121 
122 PyCharm是一种Python IDE，带有一整套可以帮助用户在使用Python语言开发时提高其效率的工具，比如调试、语法高亮、Project管理、代码跳转、智能提示、自动完成、单元测试、版本控制。
123 
124 
125 特点：
126 
127 + 编码协助
128 
129  提供了一个带编码补全，代码片段，支持代码折叠和分割窗口的智能、可配置的编辑器;
130 + 项目代码导航
131 
132  帮助用户即时从一个文件导航至另一个，从一个方法至其申明或者用法甚至可以穿过类的层次;
133 + 代码分析
134  用户可使用其编码语法，错误高亮，智能检测以及一键式代码快速补全建议;
135 + Python重构
136  在项目范围内轻松进行重命名，提取方法/超类，导入域/变量/常量，移动和前推/后退重构。
137 + ......
