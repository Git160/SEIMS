SEIMS模型数据预处理
=================

## 1.文件名约定
SEIMS模型预处理包括空间离散化（即子流域划分）、空间参数提取（土壤、土地利用等）、水文气象数据处理、最佳管理措施（BMPs）导入以及计算任务划分（MPI并行版本），因此对预处理脚本文件名进行如下约定：

+ 文件名全小写
+ 文件名应简明扼要概括代码功能，关键字用下划线分割
+ 文件名根据功能划分，加上对应前缀，前缀规定如下：
	+ 空间离散化 `sd_`
	+ 空间参数提取 `sp`
	+ 水文气象数据处理 `hydro_`
	+ 最佳管理措施导入 `bmp_`
	+ 计算任务划分 `mpi_`
