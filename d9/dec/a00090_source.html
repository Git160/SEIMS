10  Delineate hillslopes for each subbasin, include header, left, and right hillslope

12  The input stream are recommended sequenced as 1, 2, 3...
13  :param flow_dir_raster: D8 flow direction in TauDEM code
14  :param stream_value_method: method for stream cell' value.
15  0 - keep stream link code, which is default

17  2 - Set to the value of left hillslope and head hillslope
18  3 - Set stream cell to NoData
19  :return hillslope_out: If the input stream are sequenced, the output hillslope will be numbered:
20  - Header hillslope: MaxStreamID * 3 + 1
